Architecture Notes
Storage pipeline: qBittorrent downloads land in ServerSpace/Category/, arr apps hardlink into ServerSpace/Media/, Jellyfin reads from /data/Media/ inside pods via NFS mount.
Networking: All pods use hostNetwork: true with dnsPolicy: Default to work around CoreDNS instability. Workloads pinned to MachineGun via nodeSelector. GPU passthrough via NVIDIA device plugin for Jellyfin transcoding.
